Webben og planeten vår

Oppdag webbens skjulte innvirkning på planeten

Internett er helt supert, men det trenger strøm for å fungere

Store datamaskiner kalt data- sentre bruker strøm for å holde internett i gang

Elektrisiteten som brukes av nettet kan føre til noe som kalles drivhusgasser

Elektrisitet kommer fraforskjellige kilder

Ikke-fornybar energi

  • Olje
  • Kull
  • Naturgass

Fornybar energi

  • Solenergi
  • Vindkraft
  • Vannkraft
  • Geotermisk energi
Drivhusgasser forurenser planeten
  • karbondioksid (CO2)
  • metan (CH4)
  • lystgass (N2O)

Klimaendringer kan føre til:

Varmere dager

Flere varme dager kan være yucky, og det ønsker vi ikke

Naturproblemer

Det kan bli tøft for dyr og planter med det skiftende klimaet

Mindre mat og vann

Klimaendringer kan gjøre det vanskelig å dyrke mat og finne rent vann

Mennesker og helse

Varme dager kan gjøre folk syke, og vi trenger å holde oss friske

Pengeproblemer

Klimaproblemer kan koste mye penger å fikse

Helsprøtt vær

Klimaendringer gjør at stormer og skogbranner skjer oftere og blir verre

Vannstanden øker

Issmelting kan gjøre at vannstanden øker og det kan føre til flom
